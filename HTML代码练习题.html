HTML代码练习题

  1. 网站首页默认的文件名一般有( )。
    1. index.htm
    2. index.html
    3. index.asp
    4. default.htm
  2. 下列关于HTML语言的描述不正确的是( )。
    1. HTML文档是文本格式的文件
    2. HTML代码中可以嵌入多种网页特效代码
    3. 不懂HTML语言的人无法制作网站
    4. HTML语言是通过一系列特定的标记来标识出相应的意义和作用的
  3. 为了标识一个 HTML 文件，应该使用的标签是( )。
    1. <style></style>
    2. <body></body>
    3. <head></head>
    4. <html></html>
  4. 在HTML语言中，表格中行的开始和结束的标签是( )。
    1. <style></style>
    2. <body></body>
    3. <head></head>
    4. <table></table>
  5. 超文本标记语言“央视国际”的作用是( )。
    1. 创建一个指向央视国际网站的超链接
    2. 创建一个指向央视国际网站的电子邮件
    3. 插入一幅央视国际网站的图片
    4. 插入一段央视国际网站的文字
  6. 在HTML语言中，头元素的的标记是( )。
    1. <style></style>
    2. <body></body>
    3. <head></head>
    4. <table></table>
